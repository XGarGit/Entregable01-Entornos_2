# UD4 - Documentación y Refactorización

## Descripción
El programa analiza un array de números enteros, comprueba si el número máximo
se repite más de una vez y calcula la media de los valores.

## Funcionalidad
- Obtención del número máximo del array
- Comprobación de repetición del valor máximo
- Cálculo de la media
- Evaluación del resultado de la media

## Documentación
El código ha sido documentado utilizando Javadoc en la clase y en todos los métodos.

## Refactorización
Se han mejorado los nombres de variables y métodos, se ha dividido el código
en métodos independientes y se ha eliminado código innecesario para mejorar
la legibilidad y el mantenimiento.
